Список задач
Решение одной из задач является способом получить "отлично" за курс. При этом у задачи есть только два состояния: решена -- "отлично"; не решена -- надо сдавать экзамен каким-то другим способом.
Требования к решению и сдачи задачи.
  • Работа над задачей ведётся в своём форке.
  • Результат должен быть смёржен в основной репозиторий. По умолчанию в ветку dev.
  • По задаче должен быть написан технический отчёт. Отчёт должен быть оформлен в TeX по шаблону и выложен в репозиторий articles в папку 2016\fl\NameSurname (исходники и скомпилированный pdf-файл).
